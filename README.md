### 介绍
本项目依托于NUDT考研上岸人员以及个别未上岸人员的自发贡献  
致力于搜集、整理、制作考研资料，供备考同志免费使用  
本项目主体是821-Library，其他专业课的资料相对较少  
由于资源主要以PDF形式为主，所以采取管理员管理的形式
